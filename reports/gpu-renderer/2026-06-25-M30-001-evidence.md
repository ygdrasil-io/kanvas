## Changes Made

### 1. SkRRect translation (`toKanvasRRect`)
- **File:** `kanvas-skia-bridge/src/main/kotlin/org/skia/kanvas/KanvasSkiaBridge.kt`
- Added `SkRRect.toKanvasRRect()` extension function that translates SkRRect (with 4-corner radii) to kanvas `RRect` type.
- Maps all four corners via `radii(Corner)` accessor.
- Tests: `toKanvasRRect maps simple rounded rect`, `toKanvasRRect empty rect`.

### 2. SkTextBlob translation (`toKanvasTextBlob`)
- **File:** `kanvas-skia-bridge/src/main/kotlin/org/skia/kanvas/KanvasSkiaBridge.kt`
- Added `SkTextBlob.toKanvasTextBlob()` extension function.
- Added `SkTextBlob.Run.toKanvasGlyphRun()` for each run variant:
  - `HorizontalSpread` — cumulative advance via `SkFont.getWidth()`
  - `HorizontalPositions` — per-glyph x, constant y
  - `FullPositions` — interleaved positions
  - `RSXformPositions` — extract translate from xforms
- Tests: `toKanvasTextBlob from HorizontalSpread`.

### 3. KanvasSkiaBridge class
- **File:** `kanvas-skia-bridge/src/main/kotlin/org/skia/kanvas/KanvasSkiaBridge.kt`
- Created `KanvasSkiaBridge` class wrapping `Canvas` with all five draw families:
  - `drawRect`, `drawRRect`, `drawPath`, `drawImage`, `drawTextBlob`
- Tests: `KanvasSkiaBridge class wraps Canvas and delegates drawRect`, `KanvasSkiaBridge class drawRRect`, `KanvasSkiaBridge class drawTextBlob`.

### 4. SkiaKanvasSurface extended
- **File:** `kanvas-skia-bridge/src/main/kotlin/org/skia/kanvas/SkiaKanvasSurface.kt`
- Added `drawRRect` and `drawTextBlob` bridge methods.
- Delegates all draw calls to `KanvasSkiaBridge` internally.
- Tests: `SkiaKanvasSurface drawRRect records tasks`, `SkiaKanvasSurface drawTextBlob records tasks`.

### 5. Diagnostic helpers
- Added `emitBridgeDiagnostic(code, message)` and `emitUnsupportedBridgeDiagnostic(feature)` helpers.
- Emits `unsupported-skia-bridge-feature` diagnostic code for unsupported features.
- Tests: `diagnostic helper emits expected prefix`, `unsupported diagnostic helper`.

# KGPU-M31-004 — Release Notes and API Stability

## Release Notes — Kanvas GPU Renderer v1.0.0

### Summary
Kanvas is now the production default GPU renderer for the ygdrasil-io/kanvas project. The legacy gpu-raster module is frozen at M30 and will be removed in M31+.

### Supported Draw Families
- **drawRect** — axis-aligned rectangle fill and stroke
- **drawRRect** — rounded rectangle fill with per-corner radii
- **drawPath** — path fill with winding/even-odd/inverse fill types
- **drawImage** — bitmap image rendering with color type conversion
- **drawTextBlob** — text rendering via pre-shaped glyph runs (HorizontalSpread, HorizontalPositions, FullPositions, RSXformPositions)

### Shader Types
- **SolidColor** — uniform RGBA fill
- **LinearGradient** — two-point linear gradient with configurable tile mode
- **RadialGradient** — center+radius radial gradient
- **SweepGradient** — angular sweep gradient
- **Bitmap** — image-based shader with configurable tile modes

### Blend Modes
17 blend modes: Clear, Src, Dst, SrcOver, DstOver, SrcIn, DstIn, SrcOut, DstOut, SrcATop, DstATop, Xor, Plus, Modulate, Multiply, Screen, Overlay, Darken, Lighten, ColorDodge, ColorBurn, HardLight, SoftLight, Difference, Exclusion, Hue, Saturation, Color, Luminosity

### API Stability
The `:kanvas` public API surface is declared stable from this release:
- `org.graphiks.kanvas.Canvas` — @StableApi
- `org.graphiks.kanvas.Surface` — @StableApi
- `org.graphiks.kanvas.Paint` — @StableApi
- `org.graphiks.kanvas.Path` — @StableApi
- `org.graphiks.kanvas.Image` — @StableApi
- `org.graphiks.kanvas.TextBlob` — @StableApi
- `org.graphiks.kanvas.Shader` — @StableApi

Breaking changes to stable APIs require a formal deprecation cycle (one major version).

### Explicit Non-Support
- **No dynamic SkSL compilation** — runtime effects use registered Kanvas descriptors with parser-validated WGSL
- **No Ganesh/Graphite port** — WebGPU is the only GPU backend
- **No SkSL compiler/IR/VM** — SkSL is Skia API compatibility vocabulary only
- **No CPU fallback** — unsupported routes emit `RefuseDiagnostic`; no silent CPU rendering
- **No automated rollback detection** — `useLegacyGpuRaster` is a manual flag only

### Codec Support
PNG, JPEG, WebP, GIF, BMP, ICO, WBMP decode; HEIF and AVIF dependency-gated.

### Rollback Procedure
```bash
# Emergency rollback to legacy gpu-raster:
# JVM argument:
-Dkanvas.rollback.legacy-gpu-raster=true
# OR environment variable:
KANVAS_ROLLBACK_LEGACY_GPU_RASTER=true
```

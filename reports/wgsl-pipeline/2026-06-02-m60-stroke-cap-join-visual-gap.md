## Interpretation

The current WebGPU contour-fill path is close enough to be useful for
diagnosis, but it is not a support path. The next implementation slice should
start with the `round-round` case because it has the lowest regional
similarity. The follow-up should inspect the round cap arc and round join arc
coverage generated by the stroked outline before attempting to promote the
full M60 row.
